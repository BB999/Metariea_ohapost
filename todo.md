# TODO: フォールバックスクリプト修正の確認

## 2026-03-10 実施内容

### 問題
GitHub Actionsのおはツイ投稿は成功しているのに、フォールバックスクリプト(`check_and_run.sh`)が「失敗/未実行」と誤判定し、ローカルの`ohapost.sh`を実行→それも失敗→Discordに「❌ フォールバック実行失敗」が通知された。

### 原因
- macOSの省エネ設定 `networkoversleep 0` により、ディスプレイスリープ中にネットワークが切断されていた
- フォールバック実行時刻(7:10 JST)にMacがディスプレイスリープ中 → ネットワーク未接続 → `gh run list` が失敗 → 「未実行」と誤判定
- 2/20のフォールバック設置以降、18日中7回（約39%）同じ問題が発生していた

### 修正内容

1. **省エネ設定の変更**
   - `sudo pmset -a networkoversleep 1` を実行
   - ディスプレイスリープ中もネットワーク接続を維持するように変更

2. **`check_and_run.sh` のログ強化** (`local-script/fallback/check_and_run.sh`)
   - タイムスタンプ付きログ出力を追加
   - ネットワーク接続待ち機能を追加（最大60秒、5秒間隔リトライ）
   - `gh run list` のエラー出力を記録（`2>/dev/null` を廃止）
   - 各ステップの判定結果をログに記録
